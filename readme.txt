Gutenberg Block that generate AI content using Groq AI Cloud Platform

== Description ==

Automatically Write Post Summaries in WordPress with GroqAI

You can use this to write a whole post or generate a summary of the current post.

You can give commands via Groq AI Content Gutenberg Block and based on that it shows response and append into the content area.

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/groq-ai-content` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. After activating the plugin goto "Groq AI Settings" inside Settings menu and update the API Key generated from Groq Cloud Platform.
4. Now you are ready to use the "Groq AI Content" guternberg block.

== Frequently Asked Questions ==

= Does it only support GroqCloud? =

Yes, as of now it has only support to Groq Cloud

= What is the AI model it uses? =

It uses "llama3-8b-8192" model for generating content.
